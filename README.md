My website, which showcases various things about myself such as my projects, writings and photography. Houses everything and anything about myself, as well as where to find me.
## Overview
### Stack
- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Styling**: [TailWindCSS](https://tailwindcss.com/)
### Project Structure
- **`public/`**: Contains all the static assets such as images, favicons, etc.
- **`src/`**: Contains all the source code for the project.
  - **`app/`**: Contains the main application component.
  - **`components/`**: Contains all the React components used in the project.
  - **`data/`**: Contains all the data used in the project.
  - **`styles/`**: Contains all the global styles for the project.
  - **`types/`**: Contains all the TypeScript types used in the project.

## Running Locally 
This application requires Node.js v16.16.0+
```bash
git clone git@github.com:ashwnn/ashwin.lol.git
cd ashwin.lol
npm install
npm run dev
```
**NOTE**: Create a `.env` using [`.env.example`](https://github.com/ashwnn/ashwin.lol/blob/main/.env.example)
